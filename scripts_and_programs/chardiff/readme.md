# Chardiff

A tool for finding differences in the unique unordered occurrences of unicode characters in two files.

## Usage

- Run `python chardiff.py file_a file_b`.

    Replace `file_a` and `file_b` with paths to the files you would like to compare.

## Dependencies

Python 3: [Download link](https://www.python.org/downloads/)

Python `requests` module: To install it, enter the following command in cmd or a terminal:

```
pip install requests
```
